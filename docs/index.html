⚡ Bitcoin Price Prediction Dashboard

This interactive dashboard shows daily LSTM model predictions of Bitcoin prices, alongside actual BTC market prices and key financial metrics.

What it does:

  • Live daily predictions using an LSTM model trained on historical BTC data
  • Visual comparison between predicted and actual BTC prices
  • PnL tracking and market performance indicators
  • Built natively in Python (pytorch, pandas, matplotlib, numpy)

Technical Indicators Used

  • Momentum Indicators
    • RSI (Relative Strength Index)
    • Stochastic Oscillator
    • ROC (Rate of Change)
  • Trend Indicators
    • MACD (Moving Average Convergence Divergence)
    • Moving Averages (SMA, EMA)
  • Volatility Indicators
    • Bollinger Bands
  • Volume Indicators
    • On-Balance Volume (OBV)
    • Volume Moving Average

Economic Indicators Used

  • Inflation Adjusted Retruns
  • Real Interest Rate
  • GDP growth Rate
  • Unemployment Rate
  • Purchasing Power Parity

Market Indicators Used

  • Market Cap
  • Liquidity
  • Order Book Depth
  • Volume-Weighted Average Price (VWAP)
  • Network Value to Transactions (NVT) Ratio
